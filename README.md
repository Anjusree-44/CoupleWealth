# Welcome to your CoupleWealth project

# 💑 AI Couple Money Planner with Life-Event Coach

An AI-powered financial planning application designed for couples to manage money jointly, optimize taxes, plan investments, and make smarter decisions during major life events — all with multilingual and voice-enabled support.

---

## 🚀 Overview

Managing finances as a couple is complex — from balancing incomes and taxes to planning for life goals and unexpected events.

This application provides a **unified financial intelligence platform** that combines:

* Joint financial planning
* Life-event-based AI recommendations
* Tax optimization (India-focused)
* Goal-based investment planning
* Voice-enabled financial assistant

---

## 🎯 Key Features

### 👩‍❤️‍👨 Couple Money Planner

* Dual user onboarding (Partner A & B)
* Combined net worth calculation
* Joint vs individual investment optimization
* SIP allocation per financial goal
* Insurance gap analysis

---

### ⚡ Life Event AI Coach

Handles major life events like:

* 💰 Bonus
* 💍 Marriage
* 👶 New Baby
* 🏦 Inheritance

Provides:

* Smart allocation (save/invest/spend)
* Portfolio adjustments
* Tax-saving strategies
* Personalized AI explanations

---

### 🎯 Goal-Based Financial Planning

Plan for:

* 🏠 House
* 🧓 Retirement
* ✈️ Travel
* 🎓 Child Education

Includes:

* Monthly SIP calculation
* Timeline planning
* Asset allocation (Equity/Debt/Gold)

---

### 📊 Money Health Score (0–100)

Evaluates:

* Emergency preparedness
* Insurance coverage

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-couple-money-planner.git
cd ai-couple-money-planner
```

### 2️⃣ Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
